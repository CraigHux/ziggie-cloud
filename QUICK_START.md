# Ziggie Control Center - Quick Start Guide

## One-Minute Setup

### Step 1: Ensure Python is Installed
```
python --version
```
If not installed, download from: https://www.python.org (check "Add Python to PATH")

### Step 2: Create Desktop Shortcut (Optional)
Run this once to create a desktop launcher:
```
C:\Ziggie\create-desktop-shortcut.bat
```

### Step 3: Launch!
Choose one of these methods:
- **Desktop**: Double-click "Ziggie Control Center" shortcut (if created in Step 2)
- **Command Line**: `C:\Ziggie\ziggie-launcher.bat`
- **File Explorer**: Navigate to C:\Ziggie and double-click ziggie-launcher.bat

## First Time Setup Checklist

- [ ] Python 3.7+ installed
- [ ] VS Code and/or Cursor installed
- [ ] (Optional) Run `create-desktop-shortcut.bat` to create desktop shortcut
- [ ] Launch and select your preferred editor

## What the Launcher Does

1. Opens a professional GUI window
2. Let you choose between VS Code or Cursor
3. Let you select session type (New or Continue)
4. Launches your chosen editor with the C:\Ziggie workspace
5. Auto-closes after 1 second

## Troubleshooting

**Python not found?**
- Install Python from https://www.python.org
- Make sure to check "Add Python to PATH"
- Restart your command prompt

**Editor not found?**
- Install VS Code: https://code.visualstudio.com
- Install Cursor: https://cursor.sh
- Ensure installations are in default locations

**Still having issues?**
- See the full guide in: C:\Ziggie\LAUNCHER_README.md
