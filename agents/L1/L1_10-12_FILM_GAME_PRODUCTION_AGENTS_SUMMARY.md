## Integration Matrix: How the Three L1 Agents Work Together

### Workflow Sequence

#### Phase 1: Pre-Production Planning
1. **L1.12 Copywriter/Scripter** creates initial script, characters, and world
2. **L1.10 Director** reviews and provides creative direction notes
3. **L1.12** revises script based on directorial vision
4. **L1.11 Storyboard Creator** begins visual planning from approved script
5. **L1.10** reviews storyboards and provides shot/composition notes
6. **L1.11** revises boards based on director feedback

#### Phase 2: Production Preparation
1. **L1.11** finalizes production-ready storyboards
2. **L1.12** provides final shooting script with dialogue
3. **L1.10** conducts rehearsals and prepares direction approach
4. All three coordinate on technical requirements and creative vision

#### Phase 3: Production Execution
1. **L1.10** leads on-set/in-engine direction
2. **L1.11** boards serve as visual reference for all teams
3. **L1.12** available for on-the-fly dialogue revisions
4. Continuous collaboration to maintain creative vision

#### Phase 4: Post-Production
1. **L1.10** provides editing and pacing direction
2. **L1.11** creates additional boards for reshoots/revisions if needed
3. **L1.12** writes ADR dialogue and marketing copy
4. Final creative review and approval from L1.10
