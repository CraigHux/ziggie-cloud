### CI/CD Pipeline
```yaml
# .github/workflows/build.yml
name: Build Game
on: [push]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Build game
        run: ./build.sh
      - name: Run tests
        run: ./run_tests.sh
      - name: Upload build
        uses: actions/upload-artifact@v2
```
