## Typical Workflow

### Linear Film/TV Production

```
L1.12 (Script) → L1.10 (Direction Notes) → L1.12 (Revisions) →
L1.11 (Storyboards) → L1.10 (Board Review) → L1.11 (Revisions) →
Production (L1.10 Directs) → Post (L1.10 Edits, L1.12 ADR)
```

### Game Development

```
L1.12 (Narrative Design) → L1.10 (Creative Vision) → L1.11 (Cinematic Boards) →
L1.12 (Dialogue Trees) → L1.11 (Interactive Sequences) → L1.10 (Direction) →
Implementation → L1.10 (Polish Direction)
```

### Marketing Campaign

```
L1.12 (Marketing Copy) → L1.11 (Trailer Boards) → L1.10 (Creative Direction) →
L1.12 (Social Content) → L1.11 (Pitch Decks) → Delivery
```
