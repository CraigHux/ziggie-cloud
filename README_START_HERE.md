## HOW AGENTS WORK TOGETHER

### Hierarchical Coordination

```
L1 (Strategic)
  ↓ [Defines Goals & Standards]
L2 (Tactical)
  ↓ [Breaks Down Tasks & Coordinates]
L3 (Micro)
  ↓ [Executes Specific Tasks]
Result → Game Assets & Features
```

### Communication Flow

**Top-Down:**
- L1 → L2: Strategic goals
- L2 → L3: Tactical tasks

**Bottom-Up:**
- L3 → L2: Task results
- L2 → L1: Progress reports

**Horizontal:**
- L1 ↔ L1: Cross-domain coordination
- L2 ↔ L2: Pipeline integration
- L3 ↔ L3: Resource sharing

## FREQUENTLY ASKED QUESTIONS

### How many agents are in ZIGGIE?

**1,884 total agents** organized in a 12×12×12 hierarchy:
- 12 L1 Strategic Agents
- 144 L2 Tactical Agents
- 1,728 L3 Micro Agents

### What's the naming convention?

**Format:** `LX.Y.Z`
- X = Tier (1, 2, or 3)
- Y = Parent agent number
- Z = Agent number within parent (L3 only)

**Examples:**
- `L1.3` = Environment Pipeline Agent
- `L2.3.7` = L1.3's 7th L2 agent
- `L3.3.7.12` = L2.3.7's 12th L3 agent

### Where are agent files located?

All agent files are in `C:\Ziggie\agents\`:
- `L1/` - 12 strategic agent files
- `L2/` - 144 tactical agent files
- `L3/` - 1,728 micro agent files

We grew from **819 to 1,884 agents** by:
- Adding L1.10, L1.11, L1.12 (3 new strategic agents)
- Expanding each L1 from 9 to 12 L2 agents
- Expanding each L2 from 9 to 12 L3 agents
- Total increase: +1,065 agents (+130%)
